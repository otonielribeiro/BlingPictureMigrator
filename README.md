# Bling Picture Migrator\n\n## Objetivo\n\nEste projeto visa criar um software em Python para migrar fotos de produtos entre duas contas Bling (Origem: LOJAHI / Destino: SELECT) utilizando a API v3 do Bling.\n\n## Escopo\n\n- Implementação do fluxo de autenticação OAuth 2.0 para ambas as contas Bling (Origem e Destino).\n- Interface Web para inserção de SKUs dos produtos a serem migrados.\n- Lógica de download local das imagens para uma pasta temporária (`/app/data/storage/`) antes do upload para a conta de destino.\n- Upload das imagens para a conta Bling de destino.\n\n## Estrutura do Projeto\n\n```\nBlingPictureMigrator/\n├── .github/              # Arquivos de configuração do GitHub (ex: workflows CI/CD)\n├── auth/                 # Módulo para a lógica de autenticação OAuth 2.0 do Bling\n├── core/                 # Lógica principal de migração (download, upload, processamento de imagem)\n├── ui/                   # Interface Web (Flask/FastAPI/Streamlit - a definir)\n├── app/                  # Ponto de entrada da aplicação, arquivos de configuração global\n│   └── data/\n│       └── storage/      # Diretório para armazenamento temporário das imagens (volume persistente)\n├── .env.example          # Modelo para variáveis de ambiente (credenciais Bling, etc.)\n├── requirements.txt      # Lista de dependências Python\n├── README.md             # Documentação do projeto\n└── .gitignore            # Regras para ignorar arquivos no controle de versão\n```\n\n## Configuração e Instalação (Desenvolvimento)\n\n1.  **Clone o repositório:**\n    ```bash\n    git clone https://github.com/otonielribeiro/BlingPictureMigrator.git\n    cd BlingPictureMigrator\n    ```\n\n2.  **Crie e ative um ambiente virtual (recomendado):**\n    ```bash\n    python -m venv venv\n    source venv/bin/activate\n    ```\n\n3.  **Instale as dependências:**\n    ```bash\n    pip install -r requirements.txt\n    ```\n\n4.  **Configure as variáveis de ambiente:**\n    Copie o arquivo `.env.example` para `.env` e preencha com suas credenciais do Bling:\n    ```bash\n    cp .env.example .env\n    # Edite o arquivo .env com suas chaves\n    ```\n\n    **Credenciais Bling (Exemplo -
